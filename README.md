# monster-manager

### Install
```sh
git clone https://github.com/gthayer/monster-manager.git
cd monster-manager
npm install
```

### Run the app from a dev server
```sh
npm start
```

### Build the app
```sh
npm run build
```

### Host the app from a server (app must have been built)
```sh
npm run serve
```

### Switch versions (ex. v1.0.0)
```sh
git checkout v1.0.0
```
